# HYP-0033 Metals Convenience-Yield Adaptive Hazard Exit

## Design

- Uses the HYP-0031 synchronized 5-minute curve panel.
- Reuses the HYP-0032 robust detector families and entry rules.
- Adds a lagged stale-event exit: each new event looks only at previously completed same-root, same-side events for that exact variant.
- If prior events that remained open beyond a candidate age had non-positive mean net return and win rate no better than 50%, the new event exits at that adaptive age if normalization has not already occurred.
